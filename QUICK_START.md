# Быстрый старт для тестирования онбординга

## Минимальные требования для проверки онбординга

Для проверки онбординга достаточно только `BOT_TOKEN`. Остальные ключи можно добавить позже.

## Шаг 1: Установка зависимостей

```bash
pip install -r requirements.txt
```

## Шаг 2: Настройка переменных окружения

Создайте файл `.env` в корне проекта или установите переменные окружения:

```bash
# Обязательно для запуска
BOT_TOKEN=your_telegram_bot_token_here

# Опционально (для полной функциональности)
OPENAI_API_KEY=sk-...  # Нужен для обработки задач
GOOGLE_CLIENT_ID=...   # Нужен для подключения календаря
GOOGLE_CLIENT_SECRET=... # Нужен для подключения календаря
AMPLITUDE_API_KEY=...  # Нужен для аналитики
```

## Шаг 3: Запуск бота

```bash
python bot.py
```

## Что можно проверить без дополнительных ключей:

✅ **Онбординг:**
- Команда `/start`
- Выбор таймзоны (геолокация, список, ручной ввод)
- Welcome message
- Кнопка "Connect Google Calendar"

✅ **Меню:**
- Кнопки Settings и Support

❌ **Не будет работать без ключей:**
- Обработка задач (нужен OPENAI_API_KEY)
- Подключение Google Calendar (нужны GOOGLE_CLIENT_ID и GOOGLE_CLIENT_SECRET)
- Аналитика (нужен AMPLITUDE_API_KEY)

## Получение BOT_TOKEN

1. Откройте Telegram
2. Найдите @BotFather
3. Отправьте `/newbot`
4. Следуйте инструкциям
5. Скопируйте полученный токен

## Примечания

- Бот не упадет, если нет других ключей - он просто покажет заглушки
- Для полной функциональности нужно настроить все ключи (см. ENV_VARIABLES.md)
